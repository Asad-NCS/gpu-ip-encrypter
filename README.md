# Simple Usage Guide

## 1. Setup
1.  Open `config.py`.
2.  Set the IPs for your **Sender** and **Receiver** computers.

## 2. Run
You only need to run **ONE** script now:

```powershell
.\run.ps1
```

*   **On the Receiver Computer:** Select Option **2**.
*   **On the Sender Computer:** Select Option **1**.

That's it! The script will handle opening all the necessary windows for you.

## 3. Output
*   The received file will appear in the `decrypted_output` folder.
